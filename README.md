# MovieGuide

Android-приложение для просмотра информации о фильмах с использованием TMDb API.

## Технологии

- **Язык**: Kotlin
- **Архитектура**: MVVM
- **UI**: Jetpack Compose
- **Библиотеки**: 
  - Retrofit для сетевых запросов
  - Coil для загрузки изображений
  - Coroutines для асинхронных операций
  - Navigation Compose для навигации
  - ViewModel для управления состоянием

## Настройка

### 1. Получение API ключа TMDb

1. Зарегистрируйтесь на [TMDb](https://www.themoviedb.org/)
2. Перейдите в настройки аккаунта → API
3. Запросите API ключ
4. Скопируйте полученный API ключ

### 2. Настройка API ключа в проекте

Откройте файл `app/build.gradle.kts` и замените `YOUR_API_KEY_HERE` на ваш API ключ:

```kotlin
buildConfigField("String", "TMDB_API_KEY", "\"ВАШ_API_КЛЮЧ\"")
```

### 3. Сборка проекта

Проект готов к сборке. Минимальная версия Android: API 21 (Android 5.0)

## Функционал

- ✅ Главный экран со списком популярных фильмов (2 колонки)
- ✅ Экран деталей фильма с полной информацией
- ✅ Просмотр трейлеров через YouTube
- ✅ Секция "В главных ролях" с фотографиями актеров
- ✅ Обработка ошибок сети
- ✅ Индикаторы загрузки
- ✅ Поддержка русского языка

## Использование

1. Запустите приложение
2. На главном экране отобразится список популярных фильмов
3. Нажмите на фильм для просмотра деталей
4. На экране деталей можно посмотреть трейлер (если доступен)
5. Прокрутите вниз для просмотра актерского состава
